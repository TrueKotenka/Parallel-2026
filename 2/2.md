### Задача 1:
Кто первый захочет обедать (пусть Петя), звонит второму (пусть Вася)

1. Как только Петя пошёл есть - он позвонил Васе.
2. Перед тем как обедать, Вася будет ждать звонка от Пети.
3. После того как Петя пообедал, он звонит Васе.
4. Вася может идти обедать в любой момент, потому что Петя уже поел.

Аналогично наоборот.\
Всегда делаем **2 звонка** - от того, кто первый начал обедать.

Можно договориться, что приоритетом на обед обладает Петя.
Если Петя хочет обедать, и Вася ему не звонил - он идёт есть без звонка.
Как только Петя закончил, он звонит Васе.
Так можно обойтись **1 звонком** в случае, если ест сначала Петя, а потом Вася,
и они не пересекаются. 
Но в остальных случаях будет **2 звонка**.

### Задача 2:

У нас 100 потоков, каждый инкрементирует счетчик 100 раз. Операция `count = count + 1` (разбитая на чтение и запись) не атомарна.

**1. Максимальное возможное значение `count`:**\
**Ответ:** 10 000\
**Обоснование:** Максимальное значение достигается, если ни один поток не помешал другому
как при последовательном выполнении - сначала прочитал, потом прибавил и записал.

**2. Минимальное возможное значение `count`:**\
**Ответ:** 2\
**Обоснование:** Минимальное значение достигается при постоянных перезаписях.

1. Поток 1 читает `count = 0` и засыпает.
2. Поток 2 выполняет 99 итераций, теперь `count = 99`
3. Поток 1 просыпается и записывает единицу, `count = 1`
4. Поток 2 читает `count = 1` и засыпает.
5. В этот момент все остальные потоки выполняют свои операции.
6. Последний оставшийся поток 2 просыпается и пишет `count = 2`

### Задача 3:

Для 3.1 будем использовать `std::promise` для передачи сигнала (подмигивания) конкретному партнеру.

Для 3.2 можно воспользоваться барьером `std::latch(2)` При достижении барьера первый поток будет ждать, пока второй тоже дойдёт до барьера. Передадим барьер в `dance()` чтобы дождаться, пока второй партнёр тоже начнёт танцевать

```cpp
#include <iostream>
#include <mutex>
#include <queue>
#include <thread>
#include <future>
#include <latch>
#include <memory>

struct DancePair {
    // Для подмигивания
    std::promise<void> wink;
    // Барьер синхронизации
    std::shared_ptr<std::latch> sync = std::make_shared<std::latch>(2); 
};

std::mutex mtx;
std::queue<std::shared_ptr<DancePair>> leaders_q;
std::queue<std::shared_ptr<DancePair>> followers_q;

void dance(const std::shared_ptr<std::latch> &sync) {
    sync->arrive_and_wait();
    // оба партнёра готовы, можно танцевать
    std::cout << "Dancing!\n";
}

void leader() {
    std::shared_ptr<DancePair> partner = nullptr;

    // Берём в пару ведомого из очереди
    std::lock_guard<std::mutex> lock(mtx);
    if (!followers_q.empty()) {
        // Нашел ведомого
        partner = followers_q.front();
        followers_q.pop();

        // Подмигиваем партнеру
        partner->wink.set_value();
    } else {
        // Ждем ведомого
        partner = std::make_shared<DancePair>();
        leaders_q.push(partner);

        // Ждем подмигивания
        partner->wink.get_future().wait();
    }
    
    dance(partner->sync); // Выполняется строго вместе
}

void follower() {
    std::shared_ptr<DancePair> partner = nullptr;

    std::lock_guard<std::mutex> lock(mtx);
    if (!leaders_q.empty()) {
        partner = leaders_q.front();
        leaders_q.pop();

        partner->wink.set_value();
    } else {
        partner = std::make_shared<DancePair>();
        followers_q.push(partner);

        partner->wink.get_future().wait();
    }
    
    dance(partner->sync);
}

```
